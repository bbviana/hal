Badge

This component generates a small badge to the top-right of its child(ren).

Examples

Simple examples
import React from 'react';
    import Badge from 'material-ui/Badge';
    import IconButton from 'material-ui/IconButton';
    import NotificationsIcon from 'material-ui/svg-icons/social/notifications';

    const BadgeExampleSimple = () => (
  <div>
    <Badge
      badgeContent={4}
      primary={true}
    >
      <NotificationsIcon />
    </Badge>
    <Badge
      badgeContent={10}
      secondary={true}
      badgeStyle={{top: 12, right: 12}}
    >
      <IconButton tooltip="Notifications">
        <NotificationsIcon />
      </IconButton>
    </Badge>
  </div>
    );

    export default BadgeExampleSimple;

Two examples of badges containing text, using primary and secondary colors. The badge is applied to its children - an icon for the first example, and an Icon Button with tooltip for the second.

Properties

Name Type Default Description
badgeContent * node This is the content rendered within the badge.
badgeStyle object Override the inline-styles of the badge element.
children node The badge will be added relativelty to this node.
className string The css class name of the root element.
primary bool false If true, the badge will use the primary badge colors.
secondary bool false If true, the badge will use the secondary badge colors.
style object Override the inline-styles of the root element.

Other properties (not documented) are applied to the root element.

* required property
