Paper

A Paper element is a basic container that can give depth to the page.

Examples

Simple example
import React from 'react';
    import Paper from 'material-ui/Paper';

    const style = {
    height: 100,
    width: 100,
    margin: 20,
    textAlign: 'center',
    display: 'inline-block',
    };

    const PaperExampleSimple = () => (
    <div>
    <Paper style={style} zDepth={1} />
    <Paper style={style} zDepth={2} />
    <Paper style={style} zDepth={3} />
    <Paper style={style} zDepth={4} />
    <Paper style={style} zDepth={5} />
    </div>
    );

    export default PaperExampleSimple;

Paper examples showing the range of zDepth.

Properties

Name Type Default Description
children node Children passed into the paper element.
circle bool false Set to true to generate a circlular paper container.
rounded bool true By default, the paper container will have a border radius. Set this to false to generate a container with sharp corners.
style object Override the inline-styles of the root element.
transitionEnabled bool true Set to false to disable CSS transitions for the paper element.
zDepth propTypes.zDepth 1 This number represents the zDepth of the paper shadow.

Other properties (not documented) are applied to the root element.
