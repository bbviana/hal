Raised Button

This button is used to add dimension to mostly flat layouts and emphasizes important functions on your page.

Examples

Simple examples
import React from 'react';
    import RaisedButton from 'material-ui/RaisedButton';

    const style = {
    margin: 12,
    };

    const RaisedButtonExampleSimple = () => (
    <div>
    <RaisedButton label="Default" style={style} />
    <RaisedButton label="Primary" primary={true} style={style} />
    <RaisedButton label="Secondary" secondary={true} style={style} />
    <RaisedButton label="Disabled" disabled={true} style={style} />
    <br />
    <br />
    <RaisedButton label="Full width" fullWidth={true} />
    </div>
    );

    export default RaisedButtonExampleSimple;

RaisedButton with default color, primary, secondary and and disabled props applied.



Complex examples
import React from 'react';
    import RaisedButton from 'material-ui/RaisedButton';
    import ActionAndroid from 'material-ui/svg-icons/action/android';
    import FontIcon from 'material-ui/FontIcon';

    const styles = {
    button: {
    margin: 12,
    },
    exampleImageInput: {
    cursor: 'pointer',
    position: 'absolute',
    top: 0,
    bottom: 0,
    right: 0,
    left: 0,
    width: '100%',
    opacity: 0,
    },
    };

    const RaisedButtonExampleComplex = () => (
    <div>
    <RaisedButton
    label="Choose an Image"
    labelPosition="before"
    style={styles.button}
    containerElement="label"
    >
    <input type="file" style={styles.exampleImageInput} />
    </RaisedButton>
    <RaisedButton
    label="Label before"
    labelPosition="before"
    primary={true}
    icon={<ActionAndroid />}
    style={styles.button}
    />
    <RaisedButton
    href="https://github.com/callemall/material-ui"
    target="_blank"
    label="Github Link"
    secondary={true}
    style={styles.button}
    icon={<FontIcon className="muidocs-icon-custom-github" />}
    />
    </div>
    );

    export default RaisedButtonExampleComplex;

The first example uses an input as a child component. The second example has an SVG Icon, with the label positioned after. The final example uses a Font Icon, and is wrapped in an anchor tag.

Examples of Raised Buttons using an icon without a label. The first example uses an SVG Icon, and has the default color. The second example shows how the icon and background color can be changed. The final example uses a Font Icon, and is wrapped in an anchor tag.

Properties

Name Type Default Description
backgroundColor string Override the default background color for the button, but not the default disabled background color (use disabledBackgroundColor for this).
buttonStyle object Override the inline-styles of the button element.
children node The content of the button. If a label is provided via the label prop, the text within the label will be displayed in addition to the content provided here.
className string The CSS class name of the root element.
containerElement union:
 string
 element
 The element to use as the container for the RaisedButton. Either a string to use a DOM element or a ReactElement. This is useful for wrapping the RaisedButton in a custom Link component. If a ReactElement is given, ensure that it passes all of its given props through to the underlying DOM element and renders its children prop for proper integration.
disableTouchRipple bool If true, the element's ripple effect will be disabled.
disabled bool false If true, the button will be disabled.
disabledBackgroundColor string Override the default background color for the button when it is disabled.
disabledLabelColor string The color of the button's label when the button is disabled.
fullWidth bool false If true, the button will take up the full width of its container.
href string The URL to link to when the button is clicked.
icon node An icon to be displayed within the button.
label validateLabel The label to be displayed within the button. If content is provided via the children prop, that content will be displayed in addition to the label provided here.
labelColor string The color of the button's label.
labelPosition enum:
 'before'
 'after'
 'after' The position of the button's label relative to the button's children.
labelStyle object Override the inline-styles of the button's label element.
onTouchTap function Callback function fired when the button is touch-tapped.

Signature:
function(event: object) => void
event: TouchTap event targeting the button.
overlayStyle object Override the inline style of the button overlay.
primary bool false If true, the button will use the theme's primary color.
rippleStyle object Override the inline style of the ripple element.
secondary bool false If true, the button will use the theme's secondary color. If both secondary and primary are true, the button will use the theme's primary color.
style object Override the inline-styles of the root element.

Other properties (not documented) are applied to the root element.
