Floating Action Button

The Floating Action Button is used for frequently used functions.

Examples

Example
import React from 'react';
    import FloatingActionButton from 'material-ui/FloatingActionButton';
    import ContentAdd from 'material-ui/svg-icons/content/add';

    const style = {
    marginRight: 20,
    };

/**
 * Default size and `mini` FABs, in primary (default), `secondary` and `disabled` colors.
 */
    const FloatingActionButtonExampleSimple = () => (
  <div>
    <FloatingActionButton style={style}>
      <ContentAdd />
    </FloatingActionButton>
    <FloatingActionButton mini={true} style={style}>
      <ContentAdd />
    </FloatingActionButton>
    <FloatingActionButton secondary={true} style={style}>
      <ContentAdd />
    </FloatingActionButton>
    <FloatingActionButton mini={true} secondary={true} style={style}>
      <ContentAdd />
    </FloatingActionButton>
    <FloatingActionButton disabled={true} style={style}>
      <ContentAdd />
    </FloatingActionButton>
    <FloatingActionButton mini={true} disabled={true} style={style}>
      <ContentAdd />
    </FloatingActionButton>
  </div>
    );

    export default FloatingActionButtonExampleSimple;

Default size and mini FABs, in primary (default), secondary and disabled colors.

Properties

Name Type Default Description
backgroundColor string This value will override the default background color for the button. However it will not override the default disabled background color. This has to be set separately using the disabledColor attribute.
children node This is what displayed inside the floating action button; for example, a SVG Icon.
className string The css class name of the root element.
disabled bool false Disables the button if set to true.
disabledColor string This value will override the default background color for the button when it is disabled.
href string The URL to link to when the button is clicked.
iconClassName string The icon within the FloatingActionButton is a FontIcon component. This property is the classname of the icon to be displayed inside the button. An alternative to adding an iconClassName would be to manually insert a FontIcon component or custom SvgIcon component or as a child of FloatingActionButton.
iconStyle object This is the equivalent to iconClassName except that it is used for overriding the inline-styles of the FontIcon component.
mini bool false If true, the button will be a small floating action button.
onTouchTap function Callback function fired when the button is touch-tapped.

Signature:
function(event: object) => void
event: TouchTap event targeting the button.
secondary bool false If true, the button will use the secondary button colors.
style object Override the inline-styles of the root element.
zDepth propTypes.zDepth 2 The zDepth of the underlying Paper component.

Other properties (not documented) are applied to the root element.
